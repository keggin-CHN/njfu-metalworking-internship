# 所有接口记录

本文整理当前项目脚本中出现并被调用/验证过的接口（页面、AJAX、关键静态入口），用于后续维护与排障。

## 1. 说明
- 站点基址：`http://nanlin.dxsaqxx.top/`。
- 站点编码：页面与提交链路以 `gb2312/gbk` 为主。
- 文中“证据”指脚本或抓取文件中的直接调用/页面脚本片段。

## 2. 登录与门户接口

| 接口路径 | 方法 | 用途 | 关键参数/要点 | 证据 |
|---|---|---|---|---|
| `/` | GET | 打开登录页，提取 hidden 字段 | 读取 `__VIEWSTATE` 等隐藏字段 | `auto_exam.py` login 流程；`replay_login_and_fetch_pages.py` |
| `/` | POST | 提交登录 | `LoginID`、`UserPwd`、`ImageButton1.x/.y`，并配合 `USER_COOKIE` | `auto_exam.py` login 流程；`simulate_login.py` |
| `/xycms.aspx` | GET | 登录成功后门户页（frameset） | 通常由登录 302 跳转进入 | `auto_exam.py` 登录后跟随跳转；`replay_login_and_fetch_pages.py` |
| `head.aspx` | GET | 门户头部 frame 页面 | 显示“你好/用户信息”等 | `replay_login_and_fetch_pages.py` frame 抓取 |
| `left.aspx` | GET | 左侧菜单 frame 页面 | 菜单链接来源页 | `replay_login_and_fetch_pages.py` frame 抓取 |
| `huanying.aspx` | GET | 欢迎页 frame 页面 | 展示欢迎文本/状态 | `replay_login_and_fetch_pages.py` frame 抓取 |

## 3. 考试接口（正式考试 50 题）

| 接口路径 | 方法 | 用途 | 关键参数/要点 | 证据 |
|---|---|---|---|---|
| `/PersonInfo/JoinExam.aspx` | GET | 进入考试列表页 | 提取 `StartExamOne.aspx?...Start=yes` 入口 | `auto_exam.py` start_exam；`inspect_join_exam.py` |
| `/PersonInfo/JoinExam.aspx?start=yes` | GET | 考试页变体入口 | 页面 form action 可能带 `start=yes` | `PersonInfo_JoinExam.aspx_start_yes.html` |
| `/PersonInfo/StartExam.aspx` | GET | 旧入口（当前不存在） | 返回“文件不存在” | `PersonInfo_StartExam.aspx.html` |
| `/PersonInfo/StartExamOne.aspx?PaperID={pid}&UserID={uid}&Start=yes` | GET | 真正开始考试 | 从 JoinExam 页 `window.open` 提取 | `auto_exam.py`、`explore_exam_page2.py` |
| `/PersonInfo/StartExamOne.aspx?...&TestNum={irow}&NextTest=yes` | POST | 提交当前题并进入下一题 | 表单含 hidden + `AnswerN` | `auto_exam.py` 循环答题 |
| `/PersonInfo/StartExamOne.aspx?...&TestNum={irow}&PriorTest=yes` | POST | 上一题 | 页面 JS `priobtnclick()` | `start_exam_one_raw.html` |
| `/PersonInfo/StartExamOne.aspx?...&TestNum={irow}&SelTestNum={n}&SelectTest=yes` | POST | 跳题 | 页面 JS `seltestnum()/selecttest()` | `start_exam_one_raw.html`、`drill_online_learning.py` 同类模式 |
| `/PersonInfo/StartExamOne.aspx?...&TestNum={irow}&CheckAnswer=yes` | POST | 检查答卷 | 页面 JS `checkanswer()` | `start_exam_one_raw.html` |
| `/PersonInfo/StartExamOne.aspx?...&TestNum={irow}&TrySubmit=yes` | POST | 尝试交卷 | 页面 JS `trysubmit()` | `start_exam_one_raw.html` |
| `/PersonInfo/SaveExamOne.aspx` | POST | 保存答卷（中间态） | 最后一题先保存再提交 | `auto_exam.py`；`start_exam_one_raw.html` |
| `/PersonInfo/SubmExamOne.aspx` | POST | 正式交卷 | 表单提交考试结果页 | `auto_exam.py`；`start_exam_one_raw.html` |

## 7. 关键参数
- 登录表单：`LoginID`、`UserPwd`、`ImageButton1.x`、`ImageButton1.y`，并建议携带 `USER_COOKIE`。
- 考试题目字段：`AnswerN`（单选/判断通常单值；多选可重复键多值）。
- 考试上下文：`PaperID`、`UserScoreID`、`irow`、`BaseTestTypeN`、`RubricIDN` 等 hidden 字段。
- 在线学习跳题：`TestNum` + `SelTestNum` + `SelectTest=yes`。
- 视频校验：`myValue1={videoId}|{正确/错误}|{username}`。
